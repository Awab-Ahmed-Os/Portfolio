# Portfolio
Personal Portfolio Website
Tech Stack: HTML, CSS, JavaScript (Vanilla)
- Designed and built a responsive, interactive personal portfolio website showcasing projects, skills, certifications, and experience.
- Implemented animated UI elements, a multi-section single-page layout, and a certificate viewer with modal interactions -- all without external JavaScript frameworks.
- Features include: scroll-based progress bar, mobile-responsive sidebar navigation, canvas-based animated background, and a dynamic certification gallery with PDF download links.
